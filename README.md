# Jai ToDo App

Простое ToDo-приложение, созданное с использованием React, TypeScript, Vite и Tailwind.  
Разработано как тестовое задание на позицию.

## Стек технологий

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Testing Library + Vitest
- React Hook Form
- Shadcn/UI (Radix UI-based компоненты)

## Как запустить

```bash
npm install
npm run start
